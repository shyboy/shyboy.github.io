百度云突破下载速度的较好方案

Posted on 2018-05-06 | In 软件 , 教程 | | Views:
个人试用后较好的软件（命令行）

是贫穷限制了我们的消费？


是的，可是增加了我们的动手能力。。。

高速下载的2种方案

一、BaiduPCS-Go

BaiduPCS-Go 使用 Go语言编写, 为操作百度网盘, 提供实用功能.
具体功能, 参见 COMMANDS 列表

特色:
网盘内列出文件和目录, 支持通配符匹配路径;
下载网盘内文件, 支持网盘内目录 (文件夹) 下载, 支持多个文件或目录下载, 支持断点续传和高并发高速下载.

前往 https://github.com/iikira/BaiduPCS-Go 以获取更多帮助信息!
前往 https://github.com/iikira/BaiduPCS-Go/releases 以获取程序更新信息!

####第一步：下载工具：
https://github.com/iikira/BaiduPCS-Go/releases
上面链接中有各个平台的下载链接，下载后解压即可。然后双击打开BaiduPCS-Go可执行文件。

####第二步：登陆
在使用login命令的时候可能需要用到Baidu的BDUSS，可以查看这个教程。

####第三步：下载
按照提示就可以找到自己的文件进行下载了。

二、proxyee-down

使用本地http代理服务器方式嗅探下载请求，支持所有操作系统和大部分主流浏览器,支持分段下载和断点下载。
这个比较麻烦一点，可能需要进行配置。不过好了就很简单。因为是图形界面，直接在浏览器中下载后接管的。
就不多介绍了。
https://github.com/monkeyWie/proxyee-down
链接和使用教程如上。
